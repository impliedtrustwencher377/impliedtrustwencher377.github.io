1. Click the button labeled Scan.

## 🛡️ Understanding the hardening process

Hardening means reducing the surface area for attacks. The tool changes settings that programs often ignore. For example, it restricts certain system paths that AI agents might reach. It also strengthens the permissions for sensitive folders. 

You remain in control of these settings. If a feature causes minor issues with other software, you click the Revert button to return to your previous settings. The software maintains a backup of your configuration before every change.

## 🛠️ Frequently asked questions

### Does this use a lot of memory?
The tool stays in the background and uses small amounts of system resources. It only performs intensive tasks when you manually start a scan.

### Is my private data sent online?
The software performs all logic on your local machine. It does not upload your personal files or sensitive information to any cloud servers.

### How often do I run the tool?
Run the tool once each month. You should also run a scan after you install new software or change your system settings.

### What happens if the software finds a big problem?
The dashboard highlights critical issues in red. Address these items first to ensure your system stays safe.

### Can I run this alongside my antivirus? 
Yes. This tool works with your antivirus. The antivirus monitors incoming files, while Blue Spec monitors your behavior and system architecture.
